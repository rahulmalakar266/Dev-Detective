# Dev Detective

A GitHub Search Client built for Sprint 3: The API Hunter.

## Features

- Search GitHub users by username
- Fetch profile data from GitHub REST API
- Display avatar, name, bio, join date, and portfolio URL
- Show loading state while data is fetching
- Handle 404 errors with a clean "User Not Found" message
- Fetch and display latest 5 repositories
- Format ISO dates into readable dates
- Battle Mode using Promise.all()
- Compare two users by total repository stars

## Tech Stack

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- GitHub REST API
